Welcome to the challenges

These are problems that we will work on as a class to exercise the things we'll learn in the lectures. Together we'll create a component to list people. You can find that under the 'people' folder in the people.component.

Read about each challenge here:

  1. Components
  2. Built-in directives
  3. Forms and 2-way binding
  4. Composition with components
  5. Ajax
  6. Observables
  7. Services

Components

Challenge
  • Create a basic component that will read in a list of people and display only the first one.
  • You'll see a "list of people" TypeScript file. Read that into your new component.
  • Then, using interpolation, display the information about that person.
  • Apply some styles to your person and examine how the style is encapsulated.

Built-in directives

Challenge
  • Given a list of people, iterate through them and display each person. (Hint: use *ngFor).

Forms and 2-way binding

Challenge:
  • Enable 2-way binding so the user can manage their name and SSN
  • Give the user good feedback when the data entered is valid and if not, the reason why.

Composition with Components

Challenge:
  • Create a component that will display a bunch of people
  • Create a subcomponent that will live in the list component
  • Emit a delete event from the inner back up to the host. Delete the user from the host.

Ajax

Challenge: Create a component that will consume the public-facing randomUser.me API. Read about it here.

Observables

Challenge:
  • Create an async counter that updates the page every N seconds
  • Refactor the Ajax request to be an observable

Services

Challenge:
  • Create a service to share data across two unrelated components
  • Refactor the Ajax request to be made from a service instead of directly in a component
